# Setup

Ensure you have the following installed first:

- librocksdb-dev
- libclang-dev
- clang lldb lld
- build-essential
- protoc

You will also need to install rust.

To install Rust:

```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

# Editor configuration
